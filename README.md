# ♿ Campus Accessibility Navigation App

## 📖 Overview
The Campus Accessibility Navigation App is designed to assist students with disabilities in navigating campus safely and efficiently. The system focuses on accessibility-first design by providing real-time navigation, accessible route suggestions, and important campus updates.

---

## 🎯 Objectives
- Improve mobility for students with disabilities
- Provide real-time, accessible navigation
- Centralize campus facility and event information

---

## ✨ Features
- 🗺️ Real-time navigation map
- ♿ Accessible route recommendations
- 🔔 Event notifications
- 🏢 Facility discovery (e.g., restrooms, ramps, entrances)

---
